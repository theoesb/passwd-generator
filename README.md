# passwd-generator
generator of password in ruby make with love

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'passwd-generator'
```

And then execute:
 
```bash
bundle install 
```

Or install it yourself as:

```bash    
gem install passwd-generator
```
